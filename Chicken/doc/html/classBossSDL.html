Detailed Description

La classe BossSDL représente l'interface graphique du Boss dans le jeu. La classe BossSDL hérite de la classe Base et contient les méthodes nécessaires à l'affichage et à la gestion de l'objet Boss. Il contient également un vecteur d'objets BulletSDL pour gérer les balles du Boss.
