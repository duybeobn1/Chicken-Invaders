Detailed Description

Une classe pour gérer l'animation d'explosion d'un poulet dans le jeu. Cette classe hérite de la classe Base et ajoute des fonctionnalités pour gérer l'animation de l'explosion lorsqu'un poulet est touché dans le jeu. Il contient une instance de l'Explode class, un entier pour garder une trace de l'image d'animation actuelle et un tableau de SDL_Rect objets pour définir la position et la taille de chaque image dans la feuille de sprites.
