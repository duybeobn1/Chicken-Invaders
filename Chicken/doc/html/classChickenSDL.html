Detailed Description

Une classe pour gérer l'objet Chicken dans l'environnement SDL. Cette classe fournit des méthodes de gestion de l'objet Chicken dans l'environnement SDL telles que déplacer le poulet, montrer le poulet à l'écran, initialiser et manipuler les balles, etc. Il contient également un vecteur d'objets BulletSDL qui sont utilisés pour le tir par le poulet.
